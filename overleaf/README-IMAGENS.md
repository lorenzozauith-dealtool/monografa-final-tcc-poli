# Instruções para Adicionar Imagens do Aplicativo

Para compilar o documento com as imagens do aplicativo, adicione os seguintes arquivos na pasta `overleaf/`:

## Imagens Necessárias (6 no total):

1. **app-tela-principal.png** - Tela principal com mapa, status BLE e score
2. **app-telemetria-grafico.png** - Gráfico de velocidade e estatísticas
3. **app-telemetria-historico.png** - Histórico de viagens com scores
4. **app-evidencia-placa1.png** - Detecção de placa FBR2A23
5. **app-evidencia-placa2.png** - Detecção de placa HQW5678
6. **app-relatorio-viagem.png** - Relatório completo com mapa e análise

## Como adicionar:

1. Salve as 6 imagens fornecidas com os nomes acima
2. Coloque todas na pasta `overleaf/`
3. Compile o documento: `pdflatex Monografia_politex.tex`

As imagens serão incluídas automaticamente nas figuras do Apêndice B (Interface do Aplicativo iOS).
